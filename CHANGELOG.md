# Changelog

## v1.0.0 — Coming soon

Initial public release of AllOY26.

### Included

- Offline Windows desktop app
- Betfair CSV import
- World Cup group stage tracking
- Knockout tree builder
- Matchday dashboard
- Market analysis
- Position and exposure analysis
- Strategy creation and comparison
- Team, match and market notes
- PT / EN / ES language support
- Multiple currency display options
- Local-only data storage

### Notes

AllOY26 is free, offline and does not require an account.

The final installer will be published through GitHub Releases together with a SHA256 checksum.
