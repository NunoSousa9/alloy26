# AllOY26

**Free offline World Cup trading analyst for Betfair.**

AllOY26 is a Windows desktop app built specifically for World Cup trading analysis.

It lets you import Betfair CSV files, track matches and markets, analyse exposure, compare strategies and review tournament performance — all locally on your own machine.

## Key points

- Free to download and use
- Offline by design
- No account
- No cloud
- No tracking
- No subscription
- Built for the World Cup
- Supports PT / EN / ES
- Imports Betfair CSVs
